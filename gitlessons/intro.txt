GIT = распределенная система контроля версий Это система для отслеживания и ведения историй изменения файлов в вашем проеке
GitHub = веб сайт для размещения git-репозиториев в совместной разработке проектов.
Команды Git: 
1. git init = она создает новый локальный гит репозиторий у вас на пк в той дериктории в которой была приписана команда (води1 раз)тся только в начале и только 
2. git add = команда для добавление файлов в систему отслеживания гита после это комнады изменения начинают отслеживатся гитом
3. git commit -m '<comment>' = как только достигаем опредеоенного момента в разработке то тогда мы сохраняем и коментируем изменения в локальной репозитории
4. git remote add = эта команда для того что бы связать наш локальный репозиторий с удаленным репозиторием то есть добавляет подключение к репозиторий на гитхабе
5. git push = команда для отправки всех ваших изменений на локальном репозитории в удаленный репозитоорий   git push origin main
