Linux commands
ctrl=ALT+t=терминал
~ = символ домашней дерикторииinux
/= символ корневой дериктории
.. = обозначают предыдущую дерикторию
. = обозначают текущию дерикторию
1. ls (list)= команда для вывода всех дерикторий и файлов в текущей дериктории в терминале
2. cd (change directory) = команда для изменений (переходов) одной дериктории на другую cd <название дериктории>
3. mkdir (make directory) = создает указанную дерикторию в рабочей дериктории в терминале 
4. touch = создает указанный файл
5. rm = удалить файл
6. rmdir = удаление дериктории
7. pwd (present working directory) = выводитпуть до текущей дериктории
8. rm -rf = принудительное удаление
9. ls -a = выводит список всех файлов и дерикторий
10. nano <path_to_file> = откроется файл для редактирование в граффическом интерфейсе если файла нет то нано сам создаст новый файл 
11. cat <file_name> = выводит содержимое файла в терминале
 12. mkdir test test1 test2 test3 = создание нескольких файлов
 13 cd- = выйти из текущей дериктории
